# GRC-AUDIT-PROJECT
ISO 27001 compliance audit che klist and mock audit assessment
Overview
This project demonstrates practical Governance, Risk, and Compliance (GRC) skills using the ISO 27001 framework.

 Project Contents
- ISO 27001 Compliance Audit Checklist
- Mock Audit Reports
- Risk Ratings and Findings
- Remediation Recommendations

Skills Demonstrated
- ISO 27001 Compliance Auditing
- Risk Assessment
- Information Security Governance
- Control Evaluation
- Security Documentation
- Audit Reporting

 Sample Audit Areas
- Access Control
- Incident Response
- Asset Management
- Security Awareness
- Business Continuity
- Vendor Risk Management

 Tools Used
- Microsoft Excel
- Microsoft Word
- GitHub

Purpose
This project was created as part of a cybersecurity and GRC portfolio to demonstrate practical auditing and compliance capabilities.
